About

I am a passionate and dedicated learner. It's for this reason I was drawn to coding. Finding out how things work conceptually and utilizing those tools in practical hands on applications is extremely engaging and rewarding for me. I have always had a passion for technology and a drive to explore new and unfamiliar things. I have experience building dynamic web applications in JavaScript, Python and React / Redux. From my time spent bartending in some of the fastest paced bars on the eastern coast, I have learned much about teamwork, clear and concise communication, and preforming well under pressure. When not engaging my mind and continuous thirst for knowledge, I spend my free time exploring the outdoors via backpacking, rafting/kayaking, and fishing.

Projects

ThereBnB - AirBnB Clone (Solo)

This clone utilizes a python wrote flask / SQLAlchemy backend with a React / Redux frontend. It achieves its core functionality by leveraging AWS and a number of Google APIs. The bulk of the application was completed in less then two weeks, achieving all of the MVPs as well as most of the stretch goals I had laid out.

For more information please check out the github page or browse the live site.

Hoptimist - Untappd Clone (Solo)

This clone was created using a JavaScript wrote backend consisting of express and sequelize and a React / Redux frontend. I decided to put my own twist on the styling instead of going for a straight clone, while keeping all of the core functionality. The bulk of this project was completed in one week of coding, achieving all of MVPs and various stretch goals I had set out for myself.

For more information please check out the github page or browse the live site.

instagram--me - Instagram Clone (Group)

This clone was was a group project that uses a React / Redux frontend and flask / SQLAlchemy on the backend. With clean and simplistic css and the help of modals and the redux store, this application manages to grab the most of the functionality of Instagram. It leverages an AWS S3 bucket to help ease the load on the application. The majority of this project was completed in one week, achieving all of the MVPs with help of agile team planning and proper github utilization.

For more information please check out the github page or browse the live site.

TipsyStack - Stack Overflow Clone (Group)

This was a variation clone of the Stack Overflow application made with Pug and sequelize. We achieved most of the core functionality of Stack Overflow that we were shooting for by manipulating Pug Templates and CSS. This project, my first application, was done in one week, achieving all the functional MVPs that we had hoped to accomplish.

For more information please check out the github page or browse the live site.
